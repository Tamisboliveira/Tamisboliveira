## Sobre mim

Minha jornada na tecnologia começou com uma planilha.

No início da minha carreira, uma planilha importante do setor apresentou problema. Quando chamaram a pessoa responsável para resolver, descobri que era uma mulher. Aquilo me marcou muito. Ver outra mulher dominando aquela solução, explicando com paciência e resolvendo o problema com tanta segurança me inspirou a aprender também.

Comecei pelo Excel, criando planilhas para facilitar minha rotina e ajudar outras pessoas no trabalho. Depois, conheci o universo da automação com Microsoft Power Platform, onde passei a desenvolver aplicativos, fluxos e soluções low-code para melhorar processos internos.

Hoje, estou dando um novo passo: estudo Engenharia de Software para aprofundar meus conhecimentos em programação, arquitetura, APIs, frontend e desenvolvimento de sistemas mais completos.

Acredito que tecnologia não é só sobre código. É sobre resolver problemas, melhorar processos e criar soluções que façam sentido para as pessoas.

## Projetos em destaque

### RunJourney

Aplicação web em desenvolvimento para registro de corridas, com foco em prática de HTML, CSS, JavaScript e organização de dados no navegador.

Principais aprendizados:

- Manipulação do DOM
- Validação de formulários
- Armazenamento com localStorage
- Organização de código frontend
- Criação de documentação para portfólio

---

### Boa Viagem Car Rental System

Sistema de locadora de veículos desenvolvido em Python, aplicando conceitos de Programação Orientada a Objetos.

Principais aprendizados:

- Classes, atributos e métodos
- Encapsulamento
- Organização de responsabilidades
- Estruturação de projeto
- Documentação com README
  
---

### Delicinhas da Chica

Site institucional desenvolvido para uma marca de doces artesanais, com foco em presença digital, identidade visual e comunicação direta com clientes.

O projeto utiliza domínio próprio, deploy na Vercel e gerenciamento de DNS via Cloudflare.

🔗 Acesse o projeto: [delicinhasdachica.com.br](https://delicinhasdachica.com.br)

Principais aprendizados:

- Desenvolvimento de site estático
- Organização de conteúdo para pequenos negócios
- HTML e CSS
- Publicação com Vercel
- Configuração de domínio personalizado
- Gerenciamento de DNS com Cloudflare
- Aplicação de identidade visual
